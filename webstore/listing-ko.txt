짧은 설명
국가유산 보고서 PDF를 참고문헌식 파일명으로 정리하고, 인용문 복사와 e-minwon 순차 다운로드를 돕습니다.

상세 설명
📚 국가유산 보고서 PDF 이름을 연구 노트에 바로 쓰기 좋게 정리합니다.

국가유산청, 국립문화유산연구원, 국가유산 협업포털에서 발굴조사보고서 PDF를 받을 때 파일명이 download.pdf, attach_123.pdf처럼 남는 일이 많습니다. 이 확장 프로그램은 페이지에 있는 발행기관, 발행연도, 보고서 제목을 읽어 참고문헌식 파일명으로 저장되도록 돕습니다.

✨ 주요 기능
• PDF 파일명을 발행기관, 발행연도, 보고서 제목 중심으로 자동 정리
• 보고서 인용문을 팝업에서 바로 복사
• 파일명 템플릿을 원하는 순서로 조정
• e-minwon의 여러 첨부파일을 ZIP 대신 PDF별 순차 다운로드로 처리
• 툴바 팝업에서 기능 켜기/끄기
• 아이프레임 안에 있는 상세 정보도 최대한 읽어오기

🏛 지원 사이트
• 국가유산포털 간행물
• 국가유산 지식이음 발간자료
• 국가유산청 발굴조사보고서 검색/상세 페이지
• 국가유산 협업포털 발굴조사보고서 상세 페이지

🛠 0.1.2 변경
• 다른 다운로드 파일명 정리 확장 프로그램과 충돌하지 않도록 다운로드 파일명 처리 방식을 정리했습니다.
• 실제로 바꿀 보고서 다운로드가 있을 때만 파일명 결정 과정에 참여합니다.
• 빈 파일명 제안으로 다운로드 오류가 나는 상황을 피하도록 보강했습니다.

개인 연구 자료를 모을 때 파일명 손질 시간을 줄이려 만든 도구입니다. 사이트 구조가 바뀌면 일부 페이지에서 인식이 달라질 수 있습니다.
