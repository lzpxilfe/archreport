# 국가유산 보고서 파일명 정리기 (Korea Heritage Report Renamer)

국가유산 청 및 국립문화유산연구원 등의 사이트에서 유적 조사 보고서 PDF를 다운로드할 때, 웹 페이지의 메타데이터(발행기관, 발행연도, 보고서제목 등)를 기반으로 참고문헌식 파일명으로 정비하여 저장해 주는 크롬 확장 프로그램입니다. 

또한, 현재 보고 있는 보고서의 학술 인용 서지 정보(`조사기관명, 발행년도, 『보고서명』`)를 원클릭으로 클립보드에 복사할 수 있는 기능을 제공합니다.

---

## 주요 기능

1. **참고문헌식 파일명 자동 정리**
   - 기본 파일명 저장 형태: `[발행기관], [발행연도], [보고서제목] [순서번호].pdf`
   - 한 페이지에서 여러 파일을 다운로드 시 순서대로 번호(`1`, `2`, `3`...)를 자동 부여합니다. 단일 파일인 경우 번호는 생략됩니다.

2. **사용자 지정 파일명 템플릿 설정**
   - 확장 프로그램 팝업의 **파일명 설정** 탭에서 항목 칩(발행기관, 발행연도, 보고서제목, 번호, 허가번호 등) 및 구분자 타일을 드래그하여 파일명 구성을 자유롭게 변경할 수 있습니다.
   - 변경된 규칙은 브라우저 스토리지(`chrome.storage.sync`)를 통해 자동으로 동기화 보존됩니다.

3. **실시간 미리보기 연동**
   - 사용자가 보고 있는 페이지의 실제 데이터를 기반으로 가공될 파일명을 팝업창 내에서 실시간으로 미리 확인해 볼 수 있습니다.

4. **학술 인용문 원클릭 복사**
   - 팝업 내 **보고서 인용** 탭에서 현재 보고 있는 보고서의 공식 서지 인용구(`조사기관명, 발행년도, 『보고서명』`)를 제공하며, 클릭 시 클립보드로 자동 복사되어 손쉽게 한글/워드 참고문헌에 서술할 수 있습니다.

5. **아이프레임(iframe) 정보 완벽 대응**
   - 테이블이나 상세 페이지 내용이 `iframe` 태그 내에 숨겨져 렌더링되는 웹 구조에서도 백그라운드 캐시 시스템을 통해 데이터를 정상 추출합니다.

## 설치 및 사용 방법

1. 본 저장소를 다운로드하거나 복제(`git clone`)합니다.
2. 크롬 브라우저를 열고 `chrome://extensions` 주소로 이동합니다.
3. 우측 상단의 **개발자 모드** 토글을 활성화합니다.
4. 좌측 상단의 **압축해제된 확장 프로그램을 로드합니다** 버튼을 누릅니다.
5. 본 폴더를 통째로 선택하여 로드합니다.
6. 타겟 보고서 페이지로 이동한 뒤 페이지를 새로고침(F5)하여 사용을 시작합니다.
