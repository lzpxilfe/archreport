# 국가유산 보고서 파일명 정리

국가유산청 발굴조사보고서, 국가유산포털 간행물, 국가유산 지식이음 발간자료 PDF를 페이지 정보 기반의 읽기 좋은 파일명으로 저장하는 Chrome 확장프로그램입니다.

## 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 누릅니다.
4. 이 폴더를 선택합니다.

## 기본 동작

- 기본 파일명은 참고문헌에 옮기기 좋은 쉼표 기반입니다.
- 예: `전북문화재연구원, 2009, 정읍 고부구읍성.pdf`
- 기관이나 연도가 없으면 찾은 항목만 사용해 자연스럽게 줄입니다.
- `_` 구분자는 설정 화면에서 직접 배열했을 때만 사용됩니다.
- 확장 아이콘을 누르면 설정 화면이 열립니다.
- 국가유산청 발굴조사보고서 목록은 `e-minwon.go.kr` iframe 안의 상세 표와 첨부파일 정보를 읽습니다.

## 설정

설정 화면에서 항목 타일을 배열해 파일명 규칙을 바꿀 수 있습니다.

- 항목: `보고서제목`, `첨부파일명`, `발행연도`, `발행기관`, `허가번호`, `유적명`, `제출일`, `조사 시도`, `조사 시군구`, `원본파일명`, `번호`
- 구분: `공백`, `_`, `-`, 괄호, 직접 입력 텍스트
