# 📜 국가유산 보고서 파일명 정리기

> *이 프로그램을 쓰기 前에 파일명 고티기를 돌볼새, 다운로드 폴더ㅣ 어수선ᄒᆞ야 쓰기 어려우니 이를 爲ᄒᆞ야 새로 크롬 확장ᄋᆞᆯ 짓노니...*

국가유산청 발굴조사보고서 PDF를 다운받을 때, 페이지 정보를 읽어 **읽기 좋은 파일명**으로 자동 저장해주는 Chrome 확장프로그램입니다.

각 고고학연구소마다 파일명 규칙이 제각각이라 매번 손으로 고치던 그 괴로움, 이제 끝입니다.

---

## ✨ 이런 마법이 일어납니다

| 기존 파일명 😩 | 변환 후 파일명 😌 |
|---|---|
| `20090301_download.pdf` | `전북문화재연구원, 2009, 정읍 고부구읍성.pdf` |
| `file(1).pdf` | `국립경주문화재연구소, 2015, 황룡사.pdf` |
| `attach_1234567890.pdf` | `한울문화재연구원, 2021, 김해 대성동.pdf` |

---

## 🏛️ 지원 사이트

- 🏺 **국가유산청 발굴조사보고서** (`e-minwon.go.kr`)
- 📚 **국가유산포털 간행물**
- 🗂️ **국가유산 지식이음 발간자료**

---

## 🚀 설치 방법

1. 이 레포지토리를 다운로드하거나 `git clone`
2. Chrome에서 `chrome://extensions` 열기
3. 오른쪽 위 **개발자 모드** 토글 켜기
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 이 폴더 선택

> 아직 Chrome 웹스토어 등록 전입니다. 언젠간 하겠습니다.

---

## ⚙️ 기본 동작

- 📌 기본 파일명은 **참고문헌에 바로 쓸 수 있는 쉼표 기반**
  - 예: `전북문화재연구원, 2009, 정읍 고부구읍성.pdf`
- 🔍 기관이나 연도 정보가 없으면, 있는 항목만 골라 자연스럽게 줄임
- 🧩 `_` 구분자는 설정 화면에서 직접 배열했을 때만 사용
- 🖱️ 확장 아이콘 클릭 → 설정 화면 열림
- 📋 `e-minwon.go.kr` iframe 안의 상세 표와 첨부파일 정보를 직접 읽음

---

## 🛠️ 설정

확장 아이콘을 클릭하면 설정 화면이 열립니다. **타일을 드래그해서** 파일명 구성 순서를 마음대로 바꿀 수 있어요.

### 사용 가능한 항목

| 항목 | 설명 |
|---|---|
| `보고서제목` | 보고서의 공식 제목 |
| `발행기관` | 조사를 수행한 기관명 |
| `발행연도` | 보고서 발행 연도 |
| `유적명` | 조사 유적의 이름 |
| `조사 시도` | 조사 지역 시/도 |
| `조사 시군구` | 조사 지역 시/군/구 |
| `허가번호` | 발굴조사 허가 번호 |
| `제출일` | 보고서 제출일 |
| `첨부파일명` | 원본 첨부파일명 |
| `원본파일명` | 서버의 원본 파일명 |
| `번호` | 순번 |

### 구분자 옵션

`공백` · `_` · `-` · `괄호` · `직접 입력`
